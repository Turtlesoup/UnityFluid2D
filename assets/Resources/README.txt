*****************
***   SETUP   ***
*****************

Create a new game object layer called "Fluid".
Replace the main camera with the "Cameras" prefab.
Set the "Mask Camera" Culling Mask to the "Fluid" layer.
Set the "Main Camera" Culling Mask to everything else, ignoring "Fluid" layer.
Place the "Fluid Spawner" prefab into the scene and set its layer to the "Fluid layer".
